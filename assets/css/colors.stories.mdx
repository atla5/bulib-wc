# Color Scheme 

Colors are an important part of making sites look cohesive and related. 

This page exists to demonstrate the branding decisions described in the 
  university-wide branding guidelines [http://www.bu.edu/brand/guidelines/design/colors/] and how they 
  relate to BU Libraries CSS Variables [/assets/css/variables.css].

## Primary 

Colors that wrap the main content of the page, used in the header and footer.


<Story name="Primary">
  {html`
    <div class="tiles">
      <bulib-color var="--color-primary-background" white></bulib-color>
      <bulib-color var="--color-primary-background-light" white></bulib-color>

      <bulib-color var="--color-primary-text"></bulib-color>
      <bulib-color var="--color-primary-text-light"></bulib-color>
    </div>
  `}
</Story>
