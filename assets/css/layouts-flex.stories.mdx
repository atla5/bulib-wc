# Flex layouts

The `display: flex` layout is best for when you want to display things horizontally, or 
  for all-purpose responsive behavior where you want to have things wrap when their `max-width`s collide.

_NOTE: 'flex' is short for 'flexible', so this won't necessarily align items for you_

## Flex-Wrap

The default behavior for `display: flex` is to wrap (move to the next line when it overflows). 
  Notice

<Preview>
  <Story name="flex-wrap">
    {html`
      <div class="flexw bordered">
        <div class="demo">list item one</div>
        <div class="demo">list item two</div>
        <div class="demo">list item three is very long</div>
        <div class="demo">list item four</div>
        <div class="demo">list item five is even longer than item three</div>
      </div>
    `}
  </Story>
</Preview>

## Preventing Wrapping

If you want to display things horizontally, but to prevent them from wrapping, use `flexnw` to add the `flex-wrap: no-wrap`.

_NOTE: if you do this, you'll have to make sure to account for the overflow and/or any responsive behavior_

<Preview>
  <Story name="flex-no-wrap">
    {html`
      <div class="flexnw bordered">
        <div class="demo">list item one</div>
        <div class="demo">list item two</div>
        <div class="demo">list item three is very long</div>
        <div class="demo">list item four</div>
        <div class="demo">list item five is even longer than item three</div>
        <div class="demo">list item six</div>
      </div>
    `}
  </Story>
</Preview>

### Arranging Flex Items

the `justify-content` contains a number of options that will specify the arrangement:

<Preview>
  <Story name="flex-center">
    {html`
      <code>flex-center</code>
      <div class="flexw flex-center">
        <div class="demo">list item one</div>
        <div class="demo">list item two</div>
        <div class="demo">list item three</div>
      </div>
      <code>flex-between</code>
      <div class="flexw flex-between">
        <div class="demo">list item one</div>
        <div class="demo">list item two</div>
        <div class="demo">list item three</div>
      </div>
      <code>flex-evenly</code>
      <div class="flexw flex-evenly">
        <div class="demo">list item one</div>
        <div class="demo">list item two</div>
        <div class="demo">list item three</div>
      </div>
      <code>flex-around</code>
      <div class="flexw flex-around">
        <div class="demo">list item one</div>
        <div class="demo">list item two</div>
        <div class="demo">list item three</div>
      </div>
    `}
  </Story>
</Preview>


### Wrap-Direction

By default, flex will wrap in the order of the list from left to write. You can reverse the order of this 
  with `.wrap-reverse`

<Preview>
  <Story name="wrap-reverse">
    {html`
      <div class="flexw wrap-reverse">
        <div class="demo">list item one</div>
        <div class="demo">list item two</div>
        <div class="demo">list item three is very long</div>
        <div class="demo">list item four</div>
        <div class="demo">list item five is even longer than item three</div>
      </div>
    `}
  </Story>
</Preview>
