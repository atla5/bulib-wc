# Spacing

Having a consistent amount of spacing between or within elements can help bring about a 
  more pleasing and branded typology. Having a number of functional css utility classes 
  helps to simplify common spacing bugs.

## Padding 

The padding around the element makes the element itself take up more space

## Location/Orientation Options

You can add padding (`p`) or margins (`m`) to one or more sides of a given element 
  by adding classes (e.g. `.pal` = _padding all large_) with the following pattern: 

| character | attribute | meaning |
|:---------:|:---------:|:-------:|
|  `p`, `m`  | padding or margin | padding, margin |
|`a`,`v`,`h`,`t`,`b`,`l`,`r`|location| all, vertical, horizontal, top, bottom, left, right|
|`s`,`m`,`l`,`xl`, `n`| size | small, medium, large, xlarge, none|
