# Tabs

Display a variety of related content in a single pane.


## Usages

Note: the tabs should be responsive, and the input group accessible via the tab index. Once focus on 
  an input is obtained, you should be able to switch between active tabs with the arrow keys.

### Basic Example

Make sure you remember to add the `checked` attribute to the first `input`.

<Preview>
  <Story name="basic">
    {html`
      <div class="tabs">
        <input type="radio" name="tabs" id="option-1" checked>
        <label for="option-1">Option One</label>
        <div>the content within the first option</div>
        <input type="radio" name="tabs" id="option-2">
        <label for="option-2">Option Two</label>
        <div>the content within the second option</div>
        <input type="radio" name="tabs" id="option-3">
        <label for="option-3">Option Three</label>
        <div>the content within the third option</div>
      </div>
    `}
  </Story>
</Preview>

  and you can used the `checked` attribute to set a default tab.
