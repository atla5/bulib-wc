# Composites

The many helpers in this section (both functional and object-oriented), can be combined to form more complex 
  composite elements. 

The following examples contain a number of these helpers. Try pressing "Show code" on each example to find them. 
  You should be able to copy that code into anywhere that has our assets loaded and get the same result.
