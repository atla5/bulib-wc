# BU Libraries Header 
  
Provide a clean, consistent, minimalist, and reliable way to brand any given site and to navigate between subsites 
  whether on large screens or on mobile, even when `javascript` is turned off.

_NOTE: the css for this requires a build step_

Contains the main navigation across/between all sites.

## Banner

Contains Site-Specific Code/links, and normally a `<bulib-search>` (if one's not already present on the home page).

Links for a hierarchy of '**parent**: child' (unless otherwise specified), and should take you back to the site landing page (/).

<Preview>
  <Story name="site-banner">
    {html`
      <div class="banner-wrapper">
        <div class="banner">
          <span><a href="/">Ask a Librarian:</a>&nbsp; <a href="/businessFAQs">Business FAQs</a></span>
          <bulib-search str_options="pardee-help primo"></bulib-search>
        </div>
      </div>
    `}
  </Story>
</Preview>
